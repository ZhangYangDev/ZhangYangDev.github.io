Nginx 的简单操作:

关闭防火墙:

systemctl stop firewalld.service

查看 Nginx版本：

nginx -v

启动 Nginx服务：

systemctl start nginx

停止 Nginx服务：

systemctl stop nginx

重启 Nginx 服务：

systemctl restart nginx
